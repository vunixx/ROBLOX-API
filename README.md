# Vercel RoProxy Serverless

A lightweight serverless proxy for Roblox API, designed for Vercel.

## Supported Endpoints
- /v1/games/:gameId/game-passes
- /v2/users/:userId/games

## Environment Variables
- ROBLOX_API_KEY (optional)
- PROXY_KEY (optional)
